About

This portal allows the viewing and management of various aspects of silkroad inc, our made up company. The web frontend is comprised of javascript, flask, bootstrap, and many other libraries and utilities almost all of which we have tweaked in some way.

Features include user authentiction and registration, dynamic capabilities based upon the user role, viewing and editing of stores, employees, and users, and access to several quick statistics provided via plpgsql and pure sql stored procedures. The whole site is supported by a local postgreSQL database backend.
